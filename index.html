About this page

This single-file page demonstrates the minimal structure you need for GitHub Pages:

  1. An index.html at the repository root
  2. Static assets (images, CSS, JS) in the same repo or in folders like /assets
  3. GitHub Pages set to serve from the main branch (root) or the docs/ folder

How to edit

  1. Open your repository on GitHub and click Add file → Create new file.
  2. Name it index.html (or edit the existing one) and paste this HTML content.
  3. Commit the change and visit https://yourusername.github.io (or the project URL).
